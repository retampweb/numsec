# Numsec Chat Commands Protocol

Этот файл нужен ассистентам (Cursor/Windsurf/Trae и др.), чтобы они понимали, что означает команда **`/numsec "..."`** в чате.

Коротко:
- Истина проекта — в `numsec/`.
- Большие изменения оформляй через `openspec/changes/<id>/...`.

См. разделы:
- `numsec/project.md`
- `numsec/architecture.md`
- `numsec/security-requirements.md`
- `numsec/threats/THREAT-*/threat.md`

Примеры:
- `/numsec "analyze"`
- `/numsec "spec list"`
- `/numsec "spec show cli"`
- `/numsec "change new add-auth-spec"`
- `/numsec "apply add-auth-spec"`
